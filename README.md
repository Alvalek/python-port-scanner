#  Python Port Scanner

A multi-threaded port scanner built for cybersecurity education and portfolio development.

##  Description

This tool scans target IP addresses to identify open ports using TCP socket connections. It demonstrates fundamental networking concepts, Python socket programming, and multi-threading techniques.

##  Disclaimer

**This tool is for educational purposes only.** Only scan networks you own or have explicit permission to test. Unauthorized scanning may violate various national cybercrime laws.

##  Features

- Multi-threaded scanning for speed
- Configurable port ranges
- Service identification for common ports
- Clean output formatting
- Command-line argument parsing
- Error handling and timeout management
